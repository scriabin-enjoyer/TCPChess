The programs in this directory are used by me to learn about and play around
with various tools and concepts that will be relevant for implementing this
chess project. In particular, I will be exploring the following:

1. Threads and concurrent programming in Ruby
2. Working with system processes from Ruby
3. Working with Ruby's socket interfaces (TCP and Unix Domain Sockets) to
   facilitate IPC
4. Exploring the TTY toolkit gem, made by Piotr Murach.
